# Chained Animations

Una aplicación iOS que demuestra animaciones encadenadas con UIKit. La app muestra dos etiquetas de texto en pantalla (título y cuerpo) que se animan secuencialmente al tocar la pantalla. Las animaciones utilizan efectos de spring para crear movimientos fluidos: primero se desplazan horizontalmente, luego se desvanecen mientras se mueven verticalmente hacia arriba. Este proyecto sirve como ejemplo educativo de cómo implementar animaciones complejas y encadenadas en Swift.
